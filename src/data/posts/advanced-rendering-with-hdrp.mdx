# 使用 Unity HDRP 进行高级渲染

高清渲染管线 (HDRP) 是 Unity 在强大硬件上创建高保真图形的解决方案。它提供了一系列高级功能。

## 实时光线追踪

HDRP 支持实时光线追踪效果，如：

- **光线追踪反射**：准确、高质量的反射。
- **光线追踪阴影**：柔和、逼真的阴影。
- **光线追踪环境光遮蔽**：更准确、详细的环境光遮蔽。

```csharp
// 要启用光线追踪，你需要在 HDRP 资产中进行配置
// 并使用光线追踪兼容的材质和灯光。
```

## 体积光照和雾效

HDRP 允许你创建逼真的体积光照和雾效，光线与雾效交互产生可见的光束和光柱。

- **体积雾**：一种可以受灯光影响的全局雾效。
- **局部体积雾**：在场景中放置雾效体积以实现局部效果。

## 高级材质

HDRP 提供了一组新的着色器和材质，具有高级功能，如：

- **次表面散射**：用于逼真的皮肤、蜡和其他半透明材质。
- **透明涂层**：用于汽车油漆等材质。
- **各向异性**：用于具有方向性纹理的材质，如拉丝金属。
